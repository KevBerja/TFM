Machine Learning as a Service


Esta aplicación desarrollada como Trabajo Fin de Grado se hará un estudio y desarrollo de algunos de los algoritmos más utilizados en las técnicas de Machine Learning desarrollados principalmente en Python 3, junto con otros lenguajes de programación, y también basada en contenedores Docker que permite la compatibilidad en diferentes entornos o sistemas operativos y herramientas de software libre.
